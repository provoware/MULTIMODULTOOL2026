Raster:
Leerer Arbeitsbereich

Arbeitsraster mit neun Feldern. Module werden automatisch gruppiert und können per Pfeiltasten verschoben werden. Schwebende Fenster bleiben beim Vergrößern oder Verkleinern des Browserfensters im sichtbaren Bereich.

Modularer Arbeitsbereich

Der Arbeitsbereich startet leer. Aktivieren Sie links ein Modul oder erstellen Sie ein eigenes Modul. Es wird automatisch in das erste freie Feld des 3×3-Rasters eingesetzt; es wurde noch nichts gespeichert oder verändert.

1. AktivierenModule links einschalten; die Einblendung erfolgt automatisch.
2. PositionierenBei Bedarf per Drag-and-Drop in ein anderes Rasterfeld ziehen.
3. AnpassenFenster verschieben, skalieren, minimieren oder maximieren.
Modul erstellen
Pflichtfeld: kurzer sichtbarer Name des Moduls. Der Platzhalter ist nur ein Beispiel.
Optional: kurze Erklärung für die Modulübersicht. Der Platzhalter ersetzt keine Eingabe.
Konfiguration importieren
Checkliste: 1. JSON einfügen oder Datei wählen. 2. Größe und Speicherplatz prüfen. 3. Sicherheitsabfrage bestätigen. 4. Lokales Backup wird erstellt. 5. Danach importieren.
Optional, wenn eine Datei gewählt wird. Ohne gültige JSON-Daten wird nichts importiert oder gespeichert.
